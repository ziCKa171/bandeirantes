## 🚀 COMO COMEÇAR

### Para Usar o App
1. Clique no link de download (iPhone ou Android)
2. Instale o app
3. Crie uma conta ou faça login
4. Comece a usar!

### Para Desenvolvedor
1. Clone: `git clone https://github.com/ziCKa171/bandeirantes.git`
2. Instale: `npm install` (em cada pasta)
3. Configure: `.env.example` → `.env`
4. Execute: `npm run dev` (backend)
5. Execute: `npx expo start` (mobile)
